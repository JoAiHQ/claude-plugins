# Medium for Claude

Generated JoAi Claude plugin for the `medium` app.

## MCP Endpoint

`https://cortex.joai.ai/mcp/apps/medium`

## Install

Add marketplace and install:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-medium@joai-claude-plugins
```
