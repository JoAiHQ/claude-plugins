# GitHub for Claude

Generated JoAi Claude plugin for the `github` app.

## MCP Endpoint

`https://cortex.joai.ai/mcp/apps/github`

## Install

Add marketplace and install:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-github@joai-claude-plugins
```
