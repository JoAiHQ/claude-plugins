# GitHub for Claude

Generated JoAi Claude plugin for the `github` app.

## Install

Copy marketplace command:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
```


Then:

```bash
claude plugin install --scope user joai-github@joai-claude-plugins
```
