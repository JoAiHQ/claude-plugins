# Cal.com for Claude

Generated JoAi Claude plugin for the `cal-com` app.

## Install

Copy marketplace command:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
```


Then:

```bash
claude plugin install --scope user joai-cal-com@joai-claude-plugins
```
