## MCP Endpoint

`https://cortex.joai.ai/mcp/apps/http`

## Install

Add marketplace and install:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-http@joai-claude-plugins
```
