# HTTP for Claude

Generated JoAi Claude plugin for the `http` app.

## Install

Copy marketplace command:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
```


Then:

```bash
claude plugin install --scope user joai-http@joai-claude-plugins
```
