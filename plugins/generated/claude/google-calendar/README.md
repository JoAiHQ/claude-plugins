# Google Calendar for Claude

Generated JoAi Claude plugin for the `google-calendar` app.

## MCP Endpoint

`https://cortex.joai.ai/mcp/apps/google-calendar`

## Install

Add marketplace and install:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-google-calendar@joai-claude-plugins
```
