# WhatsApp for Claude

Generated JoAi Claude plugin for the `whatsapp` app.

## MCP Endpoint

`https://cortex.joai.ai/mcp/apps/whatsapp`

## Install

Add marketplace and install:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-whatsapp@joai-claude-plugins
```
