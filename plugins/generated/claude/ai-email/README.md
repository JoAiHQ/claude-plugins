# AI Email for Claude

Generated JoAi Claude plugin for the `ai-email` app.

## Install

Copy marketplace command:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
```


Then:

```bash
claude plugin install --scope user joai-ai-email@joai-claude-plugins
```
