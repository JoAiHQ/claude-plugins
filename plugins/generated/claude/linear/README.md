# Linear for Claude

Generated JoAi Claude plugin for the `linear` app.

## Install

Copy marketplace command:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
```


Then:

```bash
claude plugin install --scope user joai-linear@joai-claude-plugins
```
