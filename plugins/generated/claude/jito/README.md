# Jito for Claude

Generated JoAi Claude plugin for the `jito` app.

## Install

Copy marketplace command:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
```


Then:

```bash
claude plugin install --scope user joai-jito@joai-claude-plugins
```
