# Threads for Claude

Generated JoAi Claude plugin for the `threads` app.

## Install

Copy marketplace command:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
```


Then:

```bash
claude plugin install --scope user joai-threads@joai-claude-plugins
```
