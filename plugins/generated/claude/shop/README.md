# Shop for Claude

Generated JoAi Claude plugin for the `shop` app.

## Install

Copy marketplace command:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
```


Then:

```bash
claude plugin install --scope user joai-shop@joai-claude-plugins
```

`https://cortex.joai.ai/mcp/apps/shop`
