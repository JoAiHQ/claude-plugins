# Solana for Claude

Generated JoAi Claude plugin for the `solana` app.

## Install

Copy marketplace command:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
```


Then:

```bash
claude plugin install --scope user joai-solana@joai-claude-plugins
```
