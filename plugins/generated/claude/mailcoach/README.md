# Mailcoach for Claude

Generated JoAi Claude plugin for the `mailcoach` app.

## Install

Copy marketplace command:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
```


Then:

```bash
claude plugin install --scope user joai-mailcoach@joai-claude-plugins
```
