# Spotify for Claude

Generated JoAi Claude plugin for the `spotify` app.

## Install

Copy marketplace command:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
```


Then:

```bash
claude plugin install --scope user joai-spotify@joai-claude-plugins
```


## MCP Endpoint

`https://cortex.joai.ai/mcp/apps/spotify`
