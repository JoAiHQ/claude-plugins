# Apple for Claude

Generated JoAi Claude plugin for the `apple` app.

## Install

Copy marketplace command:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
```


Then:

```bash
claude plugin install --scope user joai-apple@joai-claude-plugins
```


## MCP Endpoint

`https://cortex.joai.ai/mcp/apps/apple`
