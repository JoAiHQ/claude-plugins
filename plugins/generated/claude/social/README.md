# Social for Claude

Generated JoAi Claude plugin for the `social` app.

## MCP Endpoint

`https://cortex.joai.ai/mcp/apps/social`

## Install

Add marketplace and install:

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-social@joai-claude-plugins
```
